1. Tổng Quan Dự Án
Relax&Chill là nền tảng web hỗ trợ sức khỏe tinh thần cho Gen Z Việt Nam, cung cấp:

💬 Anonymous Chat: Chat ẩn danh peer-to-peer real-time giữa người dùng
🤖 AI Emotional Support: Chatbot AI hỗ trợ cảm xúc hoạt động 24/7
🛡️ Zero Identity: Không yêu cầu đăng ký, không lưu thông tin cá nhân
🚨 Report System: Báo cáo nội dung tiêu cực, kiểm duyệt cộng đồng

5. Database Schema
prisma// prisma/schema.prisma

generator client {
  provider = "prisma-client-js"
}

datasource db {
  provider = "postgresql"
  url      = env("DATABASE_URL")
}

// Anonymous session — KHÔNG lưu thông tin định danh
model AnonymousSession {
  id          String   @id @default(uuid())
  sessionToken String  @unique          // JWT token
  displayName  String                   // "Anonymous#4821" — random generated
  createdAt   DateTime @default(now())
  expiresAt   DateTime                  // TTL 24 giờ
  isOnline    Boolean  @default(false)
  
  messages    Message[]
  reports     Report[]  @relation("reporter")
  
  @@index([sessionToken])
  @@map("anonymous_sessions")
}

// Chat Room
model ChatRoom {
  id          String   @id @default(uuid())
  roomType    RoomType @default(PEER)   // PEER | AI
  status      RoomStatus @default(ACTIVE)
  createdAt   DateTime @default(now())
  closedAt    DateTime?
  
  messages    Message[]
  participants ChatParticipant[]
  
  @@map("chat_rooms")
}

// Room participants (max 2 cho peer chat)
model ChatParticipant {
  id        String   @id @default(uuid())
  roomId    String
  sessionId String
  joinedAt  DateTime @default(now())
  leftAt    DateTime?
  
  room      ChatRoom @relation(fields: [roomId], references: [id])
  
  @@unique([roomId, sessionId])
  @@map("chat_participants")
}

// Messages
model Message {
  id          String      @id @default(uuid())
  roomId      String
  senderId    String?     // null nếu là AI message
  senderType  SenderType  @default(USER)  // USER | AI
  content     String      @db.Text
  createdAt   DateTime    @default(now())
  isDeleted   Boolean     @default(false)
  
  room        ChatRoom          @relation(fields: [roomId], references: [id])
  sender      AnonymousSession? @relation(fields: [senderId], references: [id])
  reports     Report[]
  
  @@index([roomId, createdAt])
  @@map("messages")
}

// Report System
model Report {
  id           String       @id @default(uuid())
  reporterId   String
  messageId    String
  reason       ReportReason
  description  String?      @db.Text
  status       ReportStatus @default(PENDING)
  createdAt    DateTime     @default(now())
  resolvedAt   DateTime?
  
  reporter     AnonymousSession @relation("reporter", fields: [reporterId], references: [id])
  message      Message          @relation(fields: [messageId], references: [id])
  
  @@map("reports")
}

// Matchmaking queue
model MatchQueue {
  id          String   @id @default(uuid())
  sessionId   String   @unique
  waitingSince DateTime @default(now())
  mood        String?  // optional mood tag
  
  @@map("match_queue")
}

// Enums
enum RoomType    { PEER AI }
enum RoomStatus  { ACTIVE CLOSED REPORTED }
enum SenderType  { USER AI }
enum ReportReason { HARASSMENT SPAM INAPPROPRIATE OTHER }
enum ReportStatus { PENDING REVIEWED RESOLVED DISMISSED }

6. API Design
Base URL: /api/v1
Anonymous Session
POST   /api/v1/session/create        → Tạo anonymous session, trả JWT
GET    /api/v1/session/me            → Lấy thông tin session hiện tại
DELETE /api/v1/session               → Xóa session (rời nền tảng)
Chat
POST   /api/v1/chat/match            → Vào queue tìm bạn chat
DELETE /api/v1/chat/match            → Rời queue
POST   /api/v1/chat/ai/start         → Bắt đầu chat với AI
GET    /api/v1/chat/rooms/:roomId    → Lấy thông tin room
GET    /api/v1/chat/rooms/:roomId/messages → Lịch sử tin nhắn (pagination)
DELETE /api/v1/chat/rooms/:roomId    → Đóng chat room
AI Chatbot
POST   /api/v1/ai/message            → Gửi tin nhắn, nhận phản hồi AI
POST   /api/v1/ai/session/reset      → Reset conversation context
Report
POST   /api/v1/reports               → Báo cáo tin nhắn
GET    /api/v1/reports/admin         → [Admin] Xem danh sách reports
PATCH  /api/v1/reports/:id           → [Admin] Cập nhật trạng thái report

7. Tính Năng & Module
Module 1: Anonymous Session System

Khi vào web → tự động tạo session UUID + JWT (expire 24h)
Lưu token vào localStorage phía client
Mỗi session có display name random: "Mây#4821", "Sao#1204"
Không thu thập email, SĐT, IP address

Module 2: Matchmaking (Peer Chat)

Người dùng nhấn "Tìm người lắng nghe" → vào queue Redis
Server poll queue, ghép cặp 2 người → tạo Room UUID
Redirect cả 2 vào /chat/[roomId]
Nếu không có người trong 60s → thông báo thử lại

Module 3: Real-time Chat

WebSocket qua Socket.IO
End-to-end: tin nhắn không lưu nếu cả 2 đều rời room (ephemeral mode)
Typing indicator ("đang gõ...")
Đọc tin / chưa đọc
Tự động đóng room sau 2h không hoạt động

Module 4: AI Emotional Support

Tích hợp OpenAI GPT-4o-mini với system prompt chuyên về lắng nghe
LangChain.js quản lý conversation memory (max 20 turns)
Phong cách phản hồi: nhẹ nhàng, không phán xét, khuyến khích
Rate limit: 30 tin nhắn/giờ (free), 100 tin nhắn/giờ (premium)

Module 5: Content Moderation & Report

Người dùng báo cáo tin nhắn vi phạm
Admin dashboard xem và xử lý reports
Auto-detect: tích hợp OpenAI Moderation API (free)


8. Bảo Mật & Ẩn Danh
✅ Không lưu IP address
✅ Không yêu cầu đăng ký / email
✅ Session UUID không liên kết với danh tính thật
✅ HTTPS bắt buộc (HSTS header)
✅ Helmet.js: X-Frame-Options, CSP, HSTS
✅ Rate limiting: 100 req/15min per IP
✅ Input sanitization: DOMPurify (client), Zod (server)
✅ JWT secret xoay 30 ngày
✅ Database: không lưu thông tin định danh cá nhân
✅ Redis cache: TTL tự động xóa dữ liệu cũ
✅ GDPR-compliant: right to deletion (xóa session = xóa data)
